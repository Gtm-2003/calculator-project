Calculator Web Application

A responsive calculator web application built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations with an interactive and user-friendly interface.

---

Features

- Addition, subtraction, multiplication, and division
- Decimal number support
- Clear display functionality
- Error handling using "try...catch"
- Responsive grid-based button layout
- Interactive UI using JavaScript DOM manipulation
- Modern calculator design using CSS styling

---

Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
